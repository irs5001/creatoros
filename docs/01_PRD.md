# CreatorOS Product Requirements Document

## Mission

Help creators convert engagement into lasting relationships and revenue through intelligent automation.

## Target Customer

Instagram creators who:
- Sell digital products
- Share affiliate links
- Want to automate comment responses

## MVP Goals

Creators can:

- Connect Instagram
- Create unlimited campaigns
- Deliver resources automatically
- Sell digital products
- Share affiliate links
- Configure a Creator Hub
- View campaign analytics

## Core Principles

1. Deliver value first.
2. Creator controls monetization.
3. Configuration over code.
4. Automation should feel human.
5. Keep the platform simple.

## Key Decisions

### Campaigns

- Unlimited campaigns per creator
- Multiple active campaigns
- Multiple triggers per campaign
- One active trigger per Platform + Content + Event
- Campaign statuses:
  - Draft
  - Active
  - Paused
  - Scheduled
  - Archived

### Triggers

A trigger is:

Platform + Content + Event

Examples:

- Instagram + Reel + Comment
- TikTok + Video + Comment
- Facebook + Post + Comment

Multiple platforms can point to the same campaign.

### Resources

- Reusable
- Multiple resources per campaign
- Resource Types:
  - Recipe
  - PDF
  - Text
  - Video
  - Link
  - File
  - Image

### Creator Hub

- One primary Creator Hub
- Modular
- Fully configurable
- Campaigns choose what appears

### Monetization

Creators decide exactly what followers receive.

Examples:

- Resource only
- Resource + Creator Hub
- Resource + Products
- Resource + Recommendations
- Any combination

The follower always receives what they requested first.

## Non-Goals (MVP)

- TikTok automation
- Facebook automation
- Mobile app
- Memberships
- Courses
- Community features
- Advanced CRM
